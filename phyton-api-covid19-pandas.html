Instalación librerías¶

En un primer momento, lo que hicimos fue instalar Pandas. Pandas es una librera de Python especializada en el manejo y anlisis de estructuras de datos. Pandas dispone de tres estructuras de datos diferentes:

  • Series: Estructura de una dimensin.
  • DataFrame: Estructura de dos dimensiones (tablas).
  • Panel: Estructura de tres dimensiones (cubos).

Para instalarlo utilizamos el motor de instalacin recomendado de Phyton que sera pip y la !, que significa que no estamos ejecutando Phyton sino Bash.
